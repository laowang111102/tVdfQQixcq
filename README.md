# 历届奥运会可视化分析系统 python758

## 项目概述

本项目是基于Python语言的历届奥运会数据可视化分析系统，融合了前端展示与后端数据处理，为用户提供了一个直观的数据查询与管理平台。

## 技术栈

- **后端：** Python 3.7.7, Django
- **前端：** Vue
- **数据库：** MySQL 5.7

## 功能模块

### 用户级别

系统将用户分为以下两个级别：

#### 一般用户

- **注册与登录：** 正常注册后可使用软件
- **基本功能：**
  - 历届奥运会数据查询
  - 运动员数据分析
  - 公告资讯浏览
- **个人中心：**
  - 编辑个人资料
  - 上传个人头像
  - 修改密码

#### 软件管理员

- **系统维护：** 管理用户、奥运会数据、运动员数据、系统信息等
- **功能权限：** 拥有一般用户的所有功能权限

## 系统角色

- **一般用户：** 使用系统基本功能
- **软件管理员：** 负责系统的管理与维护

## 运行环境

- **操作系统：** Windows 10 / MacOs
- **数据库版本：** MySQL 5.7
- **数据库可视化工具：** Navicat

## 部署步骤

1. 环境配置：安装所需Python版本、MySQL数据库、相关依赖等
2. 数据库构建：根据设计文档构建数据库结构
3. 后端部署：配置Django应用及中间件
4. 前端部署：配置Vue应用
5. 系统测试：确保各模块功能正常
